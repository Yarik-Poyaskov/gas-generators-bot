## Етап 7: Просунутий UX та Стабільність ✅
- [x] **Smart Sorting Delay**: плашки не змінюють позицію перші 10 секунд при зміні статусу.
- [x] **Extended Flashing**: миготіння плашки збільшено до 15 секунд (10 до + 5 після переміщення).
- [x] **Anti-Sleep WebSocket**: автоматичний реконнект та Heartbeat для роботи на ТВ 24/7.
- [x] **Auto-Session Refresh**: продлення сесії при активності користувача та фонове оновлення для ТВ.
- [x] **Visibility Guard**: миттєве оновлення даних при поверненні на вкладку браузера.

## Етап 8: Компактний Дашборд та Безпека ✅
- [x] **GPUCardCompact**: Новий вид плашок зі зменшеною висотою та випадаючими списками.
- [x] **Smart Indicators**: Кольорова індикація наявності графіка, звіту та активної зміни на плашках.
- [x] **Admin Online Monitor**: Відображення списку користувачів, що знаходяться онлайн, у шапці (лише для адмінів).
- [x] **Role-Based Access Control**: Сховано розділи звітів від звичайних користувачів у Sidebar.
- [x] **API Shift Data**: Додано отримання ФІО та телефону чергового прямо в об'єкти дашборду.
